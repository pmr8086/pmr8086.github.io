  • Bibliography
  • Goldstein's Classical Mechanics. Standard text in classical mechanics.
  • Whittaker's Treatise on Analytical Dynamics. § 69 contains an analytic solution for the Euler top. Beware that the convention for the Euler angles is different (§ 10).
  • NIST Digital Library of Mathematical Functions (DLMF). Standard reference for special functions.
  • Whittaker & Watson's Modern Analysis. The ultimate source in special functions. Not just a reference, contains proofs.
  • Borwein's π and the AGM. Elliptic functions, the AGM and associated transformations, and their applications in the computation of π and other quantities, with computational complexity analyses.
  • Adnan Ademovic's WebGL tutorial. A lot of our code follows the examples here closely.
  • MDN WebGL tutorial. Standard source for everything web development.
  • Song Ho's website. This website contains a lot of very great material on the fundamentals (OpenGL, computer graphics in general). There are for example software demonstrations which really help visualizing the OpenGL matrices.
